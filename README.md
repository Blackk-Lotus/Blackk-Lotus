## About Me

I build open-source security tools and research the intersection of AI and cybersecurity — from threat detection to LLM vulnerabilities to blockchain security.

- 🔍 **Threat Detection** — tools that catch malware, backdoors, and C2 beaconing
- 🤖 **AI Security** — LLM prompt injection, jailbreak research, AI-powered GRC
- 🔐 **GRC** — FedRAMP, compliance automation, risk frameworks
- ⛓️ **Blockchain Security** — smart contract auditing, Web3 threat modeling
